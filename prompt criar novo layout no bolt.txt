Prompt atualizado:
"Crie um site estático em HTML PURO para um blog de decoração.

Requisitos:

Apenas arquivos: index.html, categoria.html, artigo.html, style.css, script.js

Header com logo e menu de categorias

Hero section com título e descrição (apenas na home)

Grid de cards de posts (com imagem, título, data, categoria, link)

Sidebar com posts relacionados

Footer com links e redes sociais

Design limpo, minimalista, tons quentes (bege, marrom, off-white)

Totalmente responsivo

Não use frameworks (React, Vue, Astro, etc)

Apenas HTML + CSS + JS puro

Páginas:

index.html - Página inicial

Header completo

Hero/Banner

Grid com 6 cards de exemplo

Sidebar com posts relacionados

Footer

categoria.html - Página de categoria

Header completo

Título da categoria

Grid com 4 cards de exemplo

Footer (sem sidebar)

artigo.html - Página de artigo individual

Header completo

Título do artigo

Meta (data, autor, categoria)

Imagem principal

Conteúdo do artigo (com headings, listas, tabela, FAQ)

CTA (botão de compra)

Artigos relacionados (3 cards)

Footer

Cards devem ter: imagem, título, categoria, data e link 'Ler mais'.

Use placeholders como:

{{TITULO_ARTIGO}}, {{CONTEUDO_ARTIGO}}, {{CATEGORIA}}, {{DATA}}

{{LINK}}, {{IMAGEM}}, {{AUTOR}}

Os placeholders são para um gerador dinâmico substituir depois."
